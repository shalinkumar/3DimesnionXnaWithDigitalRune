How to get the source code
--------------------------
The source code of some DigitalRune libraries is available for free if you 
have purchased an Indie or a Professional license. For more information see:
  http://www.digitalrune.com/Downloads.aspx > Premium Downloads

If you have purchased a Professional license, you can license the full 
source code for an additional fee. For more information please contact: 
  office@digitalrune.com
